Try inputmode attribute to suggest keyboard type on mobile: inputmode="numeric"

Use <script type="module"> for modern JavaScript with imports
Try defer attribute on scripts to load without blocking HTML parsing
Use async attribute on independent scripts to load and execute ASAP
